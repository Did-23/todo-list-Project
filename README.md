# 📝 Todo List

## 📌 Сипаттама
Бұл жоба — күнделікті тапсырмаларды басқаруға арналған веб-қосымша.  
Қолданушы тапсырмаларды қосып, өшіріп, орындалған деп белгілей алады.  

---

## 🧱 Құрылым және методология
Жоба **BEM (Block, Element, Modifier)** әдісі бойынша құрылды, бұл кодты түсінікті, масштабтауға ыңғайлы етеді.

### Мысал:
- Block: `todo`  
- Element: `todo__item`  
- Modifier: `todo__item--done`  

### Жоба құрылымы:
/index.html # Негізгі бет
/scss # Стильдер
/js # JavaScript логикасы

---

## 🚀 Мүмкіндіктер
- ➕ Жаңа тапсырма қосу  
- ❌ Тапсырманы өшіру  
- ✅ Тапсырманы орындалды деп белгілеу  
- 🔄 Бет жаңартылғаннан кейін тапсырмалар сақталады

---

## 🛠️ Қолданылған технологиялар
- HTML5 / SCSS  
- JavaScript (ES6+)  
- BEM методологиясы  
- Git (версия бақылау)

---
